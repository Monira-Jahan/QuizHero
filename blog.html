Question-1 : What Is the difference between map,forEach,filter and find?

Difference Between map,forEach,filter and find are-
Map(): When the specified callback to map is finished, the elements of the original array are modified and saved into the new array, and the original array is then returned as the mapped array.
forEach():Foreach accepts a callback function and executes it on each individual array element.
filter():ForEach and filter differ primarily in that forEach just loops through the array and executes the callback, whereas filter executes the callback and checks the return value.
forEach():The value of the first element that passes a test is returned by the find() method, which runs a function on each element of the array.

Question-2 : What is difference between local storage and session storage?

Answer: The difference between local storage and session storage is-
1. Local storage stored data in browser without any expiration date. Session storage data in browser with expire.
2. In local storage data is not deleted when the browser is closed. In session storage is deleted when the browser is closed.
3.In local storage data is available for future sessions. In session storage, data is not available for future session.

Question-3 : What Is event-loop?

Event-loop is -The event loop in JavaScript is a technique for moving the calls in the callback queue that are waiting to be executed to the call stack. The call stack must be empty in order for any event from the callback queue to arrive there.
